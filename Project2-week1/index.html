Interactive Elements

In-Text Link

.link-class

Code:

<a href="View More">View More</a>

View More

CSS Selector:

.interactive-button

Code:

<a class="button-class">BUTTON</a>

BUTTON

Bulletted List

CSS Selector:

li

Code:

<ul><li>First list item</li><li>Second list item</li><li>Third list item</li></ul>

Displays as:

  • First list item
  • Second list item
  • Third list item

Numbered List

Code:

<ol><li>First list item</li><li>Second list item</li><li>Third list item</li></ol>

Displays as:

  1. First list item
  2. Second list item
  3. Third list item
