Memeorabilia Style Guide

In Text Link

CSS Selectors:

none

Code:

<a href="link.html">Link</a>

Displays as:

Link

Navigation Link

CSS Selectors:

navigation-link

Code:

<div class="navigation-link">
<a href="link.html">Link</a>
</div>

Displays as:

Link

Call-To-Action Buttons

CSS Selectors:

button-class

Code:

<a class="button-class" href="link.html">Link</a>

Displays as:

Link

Forms

CSS Selectors:

form-input, labelstuff, submit

Code:

<div class="submit">
<form method="POST" action="#">
<label class="form-label" for="form-name"><a class="button-class">Button</a></label>
<input class="form-label" id="form-name" type="text" placeholder="Placeholder">
</form>
</div>

Displays as:

Text Styling

CSS Selectors:

none

Code:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<p>Paragraph</p>

Displays as:

Heading 1

Heading 2

Heading 3

Paragraph

<div class="products">

</div>
</div>
